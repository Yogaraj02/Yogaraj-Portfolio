# Yogaraj S - Cybersecurity Portfolio

Welcome to my personal cybersecurity portfolio website! This project is a visually stunning, responsive, and highly optimized single-page portfolio designed to showcase my skills, projects, certifications, and passion for cybersecurity.

## Features

- **Premium Glassmorphism Design:** Modern aesthetic featuring a dark theme, ambient animated background blobs, and frosted-glass panels (`backdrop-filter`).
- **Responsive Layout:** fully functional and perfectly scaled across desktop, tablet, and mobile devices.
- **Optimized Performance:** Extracted large embedded assets to drastically reduce HTML file size, ensuring lightning-fast load times.
- **Dynamic Interactions:** Smooth hover animations and modal integrations for a polished user experience.
- **Accessible Text Contrast:** Carefully calibrated text colors against frosted backgrounds adhering to WCAG standards.

## Technologies Used

- HTML5
- Vanilla CSS3 (Custom properties, Keyframes, Flexbox, CSS Grid)
- JavaScript (for Modal logic)

## Installation and Usage

To run this project locally, simply clone the repository and open the HTML file in any modern web browser.

```bash
git clone https://github.com/Yogaraj02/Yogaraj-Portfolio.git
cd Yogaraj-Portfolio
```

Open `Final Portfolio.html` in your browser.
